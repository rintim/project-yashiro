# Project Yashiro

> “世界是个巨大的夜白”

《夜白项目》是一系列与无名杀相关的项目，包含无名杀扩展和辅助开发无名杀的工具

名称来源于无名杀著名扩展开发者: [@nigthmoon](https://github.com/nigthmoon)

# 杂项

## 为什么中文名是《夜白项目》/ 英文名是*Project Yashiro*

-   确实，根据前人的翻译，中文名应该是《夜白计划》，不过为了体现生草感，直接直译成《夜白项目》感觉会更好
-   *Yashiro*来源于“夜“”白“两个字的日语读音组合，正经组合应该读成*Yahaku*，不过同样为了体现生草感，选用了比工地日语还工地的读法
    - 然后*Yashiro*这个读音对应的词比想象中的多，单论搜索结果还不如*Yahaku*，失算了
